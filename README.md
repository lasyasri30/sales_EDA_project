# SQL-Based Exploratory Data Analysis on Sales Data


## 📌 Executive Summary
This project performs Exploratory Data Analysis (EDA) on a retail sales dataset using SQL. The analysis focuses on understanding sales trends, customer behavior, and product performance across categories such as bikes, accessories, and clothing. SQL joins and aggregate functions were used to extract meaningful insights from structured data.

## ❓ Problem Statement
Retail data contains valuable insights, but without proper analysis, it is difficult to identify patterns in sales, customer behavior, and product performance.

## 🎯 Solution
The problem was addressed by performing EDA using SQL. By applying joins and aggregate functions, the dataset was explored to identify patterns, trends, and relationships between sales, customers, and products.

## ⚙️ Methodology
- Explored database structure and tables  
- Used SQL joins to combine multiple tables  
- Applied aggregate functions (SUM, AVG, COUNT)  
- Performed dimension and date-based analysis  
- Identified top and bottom performers  

## 🧠 Skills
- SQL (Joins, Aggregations)  
- Exploratory Data Analysis (EDA)  
- Data Analysis  
- Problem Solving  

## 📊 Results
- Identified top-performing products  
- Analyzed customer behavior  
- Observed sales trends over time  
- Compared product categories
